# 🖤 KULT — FOR THE FEW.

A **fictional luxury streetwear e-commerce website** built as part of a company assignment, focused on delivering a **cinematic, high-end user experience** with smooth animations and modern UI design.

---

## ✨ Overview

KULT is not just a basic e-commerce UI — it’s designed to feel like a **premium fashion brand website**, combining:

- Minimal luxury aesthetics  
- Smooth transitions & animations  
- Interactive product browsing experience  
- Mobile-first responsive design  

---

## 🚀 Features

- 🎬 **Cinematic Loader** (flip animation)
- 🎯 **Hero Section with Video Background & Effects**
- 🧭 **Interactive Category Split (MNS / WMNS)**
- 🔄 **Page Flip Transitions**
- 🛍️ **Product Listing with Horizontal Carousel**
- 🧾 **Product Detail Page with Image Switching**
- 🖱️ **Custom Cursor Interaction**
- 📱 **Fully Responsive (Mobile First)**
- 🎨 **Minimal Luxury UI (Black + Off-white theme)**

---

## 🛠️ Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **Framer Motion** (animations)
- **Lenis** (smooth scrolling)
